To run type:
py hw3.py [number of iterations] [number of generated chromosomes] [file with sack size and items]
Example:
py hw3.py 40 10 sample.txt

File constraints:
First line of the file must be the knapsack size and a positive integer.
The next lines of the file must be the sizes of the items and a positive integer.

5		<-- Item size
